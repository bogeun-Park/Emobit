## Emobit - 감성 일기장 웹 서비스

Emobit은 감정(Emotion)의 조각(Bit)들을 기록하고 공유하는 감성 일기장 웹입니다.



### 주요 기능
- 회원가입 및 로그인
- 게시글 작성 및 수정/삭제
- 댓글 작성 및 수정/삭제
- 조회수 중복 처리 및 캐싱
- 이미지 업로드
- 실시간 채팅 및 알림
